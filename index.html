Instructions

Souris :
  • Glisser pour orbiter
  • Molette pour zoomer
  • Clic gauche : Placer un bloc
  • Clic droit : Supprimer un bloc
Clavier :
  • WASD / Flèches : Déplacer
  • Q/E : Monter/Descendre
Tactile :
  • Glisser un doigt : Orbiter
  • Pincer/Écarter : Zoomer
  • Appuyer : Placer un bloc
  • Appuyer long : Supprimer un bloc
